        - 출석율(로그인 출석 기록)
        - 학습 성과( 테스트 / 점수 등)

        - 제출률
        - 정답률

        - 성과

            - 다음 상담 일정 리마인더 추가
            - 주기 설정

                - 월별 : 매달 [입력: ] 일 마다 / OO 시
        - 상담 성과
            - 상담 자기 평가
                - 친절도
                - 친밀도
                - 파악
                - 상담 시간
        - 학생 상담 성과 ( 최소 20자 )
